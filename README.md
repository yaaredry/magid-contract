# חוזה שכירות – מרכז קהילתי מגיד

ממשק אינטרנטי למילוי אוטומטי של חוזה שכירות.

## שימוש

### מומלץ – GitHub Pages
הפעל את GitHub Pages על הרפוזיטורי (Settings → Pages → Branch: main).  
הכלי יהיה נגיש בכתובת: `https://yaaredry.github.io/<repo-name>/`

### מקומי (Local)
לא ניתן לפתוח `index.html` ישירות דרך הדפדפן (חסימת `file://`).  
יש להריץ שרת מקומי פשוט:

```bash
# Python (מותקן בכל Mac/Linux)
cd /path/to/project
python3 -m http.server 8080
```
ואז לפתוח: `http://localhost:8080`

## כיול קואורדינטות

הטקסט מוטבע בחוזה לפי קואורדינטות X/Y המוגדרות בקובץ `index.html`.  
אם טקסט מופיע במקום לא נכון – ערוך את הבלוק `const C = { ... }` בסקריפט.

**שיטת מדידה:**
1. פתח את `contract.pdf` ב-Adobe Reader.
2. השתמש בכלי המדידה כדי לאתר כל שדה.
3. Reader מציג קואורדינטות מפינה עליונה-שמאלית, אז:  
   `y (pdf-lib) = 842 − y (Reader)`

**יחידות:** 1 נקודה = 1/72 אינץ'. דף A4 = 595 × 842 נקודות.

## עדכון החוזה

אם תבנית החוזה משתנה:
1. החלף את `contract.pdf` בגרסה החדשה.
2. כייל מחדש את הקואורדינטות אם הטקסט זז.
